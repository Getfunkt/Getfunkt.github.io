Mineral Recon

Mineral Recon is a gamejam project i made with my brother for Ryan Laley Games Summer Jam 2023 with a theme of contrast and a timelimit of 48 hours.

Link to the gamejam project Mineral Recon.

Mineral Recon was made with Unreal Engine 5.3 and models were made with Blender 3.2.

In the game player pilots a drone with a drill and a weapon with the objective to mine as many minerals they can while also avoiding or shooting threats on an asteroid. The drone moves on the asteroid like a spider clinging to the surface. The theme contrast is used as a gameplay mechanic that works by making the player adjust the contrast with the mouse wheel to keep the visibility clear while moving on the asteroid.

What i did in the project:

Visuals and the logic for the User Interface. Below is an image showcasing the UI

Weapon and drill functionality for the player

3D models and animations made with blender.

Game Engine courses 1 & 2

In the first course, the goal was to create a game engine and develop three example games based on Arkanoid, Pac-Man, and Donkey Kong.

Tools used for the projects:

Visual Studio 2019

CMake 3.5

C++

Libraries:

SFML 2.6.0

OpenGL

In the first course I made a working Arkanoid clone with features such as:

- Player-controlled paddle with left/right movement

- Ball physics with collision handling against objects

- Two playable levels

- Score system and power-ups affecting the ball and paddle

- Main menu and a pause menu

Gameplay

In the second course, my- and other team were combined so my partner and I changed to the other team's game engine because it was more developed. The goal in this course was to port the games to work both with SFML and OpenGL rendering.

I took on the other teams Pac Man clone game as it was not finished and only had movement and collisions for the ghosts and the player. I finished the game by adding:

- A* pathfinding for the ghosts.

- Turn buffer for the player.

- Fixed collisions.

- Added a second level.

- Changed SFML dependencies to work with both renderers.

VR Store Game

VR Store Game is an game made by me for an school course where the idea was to make an educational game in VR environment.

.

The project was made with Unreal Engine 5.4 blueprints and developed with both HTC Vive- and Valve Index headsets

Project overview:

VR Store is a VR game about managing a store. The main tasks include restocking shelves, guiding customers to products, scanning, and handling payment at the cash register.

Main Features:

Grid based dynamic shelving for different kinds of products for easier interaction.

Customer AI mimicking real behavior for gathering products and paying for them. Sometimes they will require guidance from the player to find what they need.

Products were implemented with data driven design. A data table is used for easier management and addition of new ones. A single base blueprint is then used for spawning them in.

Also implemented is a functional cash register that you could see in any store. You scan products and get the price readout. Includes a change calculator as well.

I learned from this project to develop VR projects using Unreal Engine and developing modular systems such as product blueprint and dynamic shelves.
